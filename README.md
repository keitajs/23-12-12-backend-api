# Backend API

## Install

```
npm install
```

## Start

Normál indítás
```
npm run start
```

Fejlesztői indítás
```
npm run dev
```

## Database

Tartalmazza az adatbázis létrehozását (táblákat a Sequelize létrehozza).
```
./sql/513szoft_zarodolgozatok.sql
```

## dotenv

Tartalmazza a .env szükséges adatait, localhoston alkalmazható a jelenlegi tartalmával.
```
.env.example
```
